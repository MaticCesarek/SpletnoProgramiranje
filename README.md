e-Študent
-------------
Izdelujem spletno aplikacijo za eletronski index študentov. Aplikacija bo reševala probleme, ki so povezani z izpiti.
-Prijava/odajava izpita
-razpisovanje izpitnega roka
-vnos ocen za posamezni izpitni rok
-hramba in pregled ocen opravljenih izpitov
-pregled eletronskega indexa
-pregled izpitnih rokov z odprtimi prijavami


Cilja publika
--------------
-Študenti
-Profesorji
-Študenski referat

Aplikacija bo delovala na raèunalnikiuh,tablicah in telefonih
